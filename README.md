## Run Locally

Clone the project

```bash
git clone https://github.com/SAHILDUDHAL21/Just-Type.git
```

Go to the project directory

```bash
cd Just-Type
```

Install dependencies

```bash
flutter pub get
```

Run the Application

```bash
flutter run <device>
```
